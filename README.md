# desktop_ai_assistant (Cluely Clone)
A desktop ai assistant app that assists users in their day to day work, from understanding text on the screen to providing instructions on how to run a program. 

The program takes screenshots of the user's screen, feeds it to a llm model, takes instructions and provides helpful output to the user.
